# Contributor Covenant Code of Conduct

## Our Pledge

We want Agent Remote Bridge to be a welcoming and professional project for users and contributors across languages, backgrounds, and experience levels.

## Our Standards

- Be respectful and constructive.
- Assume good intent, but communicate clearly.
- Focus feedback on code, behavior, and documentation, not people.
- Avoid harassment, discrimination, and personal attacks.

## Enforcement

Project maintainers may remove comments, lock threads, or reject participation that harms the health of the community.

## Reporting

Report conduct issues privately using the contact path described in [SECURITY.md](./SECURITY.md) or by opening a maintainer contact request in [SUPPORT.md](./SUPPORT.md).
