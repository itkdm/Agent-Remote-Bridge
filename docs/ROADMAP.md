# Roadmap / 路线图

## Production-candidate baseline / 生产候选基线

- Freeze the stable tool contract after one controlled cleanup pass
- Maintain green CI across Windows, macOS, and Linux
- Publish installable releases to PyPI
- Keep release notes, docs, and CLI behavior aligned

## Near-term priorities / 近期重点

1. Security boundary hardening
2. Session lifecycle enforcement
3. Reliable packaging and release automation
4. Clear contribution and support workflows

## Not in scope yet / 暂不纳入

- Interactive shells or TTY bridging
- Non-Linux remote targets
- WinRM, Docker, or Kubernetes adapters
- Multi-tenant RBAC or team control plane
